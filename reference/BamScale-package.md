# BamScale: Bioconductor-Friendly Multithreaded BAM Processing

Multithreaded sequential BAM processing built on top of the ompBAM C++
engine. BamScale provides user-friendly BAM read and scan interfaces
designed for compatibility with existing Bioconductor workflows.
